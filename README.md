# S2-DW-WOF Workflow
Google Earth Engine (GEE) workflow to map monthly and annual Water Occurrence Frequencies (WOF) from full Sentinel-2 time series. The worklfow can be applied to any region of interest and enables exporting as raster file to Google Drive.

Developed by Leo Helling

leo.helling@ens-lyon.fr UMR 5600 – Environment, Ville, Société (EVS) lab Centre National de Recherche Scientifique (CNRS) ENS de Lyon France
